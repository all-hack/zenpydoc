Setting up our computer

First, before we do anything we have to make sure that our Mac system software is up to date. The next step is to check if we have JDK 6 (Java Developer Kit) or higher installed, we can do this by typing the command javac -version into a terminal window. If the appropriate JDK is not installed we will have to download it.

What we need is the Mac OS x64 version of the JDK, to download it first click the icon that says Java, then accept Oracle's license agreement and then click the download link associated with the Mac OS x64 version of the JDK. A .dmg file will then be downloaded and put into your browser's default download folder, it should be called something along the lines of jdk-8u45-macosx-x64.dmg. Double click the .dmg file to start the installation process. Installing the JDK should be very simple just follow the instructions provided by the installer and let your computer do all the work. Once it's finnished you can double check that everything was successful by typing javac -version once more in the terminal.
