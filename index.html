lists

lists are the workhorse of python, essentially they are an indexed data structure that can hold elements of any datatypes, who's size can be changed dynamically on the fly. More detailed documentation provided at docs.python.org


Basic Python Functions for List objects

Function Call What it does
list.extend([list2]) Adds all the elements in [list2] to the end of list in order that they appear in [list2]
list.append(object) Adds object to the end of the list
list.insert(index, object) Inserts object at position index in the list
list.remove(object) Removes the first element from the list who’s value == object; produces a ValueError if object is not in list
list.pop([object]) Removes the element at index in the list and returns it. If no index is provided removes and returns the last element in the list (the [ ] indicates that the argument within the [brackets] is optional)
list.index(object) Returns the index of the first element in the list who’s value == object; produces a ValueError if object is not in list
list.count(object) Returns the number of times object appears in the list
list.sort([reverse= boolean_value]) Sorts the elements of the list; optional parameter reverse is set by a boolean_value determining if the list is sorted backwards or not. There are other optional parameters (cmp, key) to customize sorting, read full documentation for more info.
list.reverse() Reverses the order that the elements appear in the list
